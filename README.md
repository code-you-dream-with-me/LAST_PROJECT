### 1. 프로젝트명
``` 
- Intery(인테리) : 1인 가구 맞춤 인테리어 정보 공유 및 가구&소품을 제공하는 원스톱 인테리어 플랫폼
```

### 2. 목적
``` 
- 1) 온라인 집들이로 다양한 인테리어 공유 콘텐츠 제공
- 2) 구매까지 한 번에 이루어질 수 있는 원스톱 플랫폼을 구축하여 편리한 구매 경험 제공 
```
